# Netflix Clone

### Key Features:

- Nextjs App Router
- Supabase Database
- Styling with Tailwind and Shadcn UI
- Next-Auth Authentication
- OAuth (Google and GitHub)
- Passwordless Auth
- Prisma ORM
- Deployment to Vercel

### Technologies Used: 

- Next.js
- Supabase
- Prisma
- Next-Auth
- Tailwind
- Shadcn UI
- Typescript
- Vercel
